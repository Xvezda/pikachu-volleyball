Pikachu Volleyball is a classic game which was developed in Japan in 1997. This game was popular in Korea when I was young. I had played it with my friends too.

The original game can be executed only in Windows environment.

This web version is created by reverse engineering the machine code of the original game and implementing it to JavaScript. It is executed directly on the web browser so you can play it not only in Windows, but also in macOS, Linux, etc. — only if you have keyboard:) You can view the source code on GitHub.

If you select "1人でぴかちゅ~" with "Z" key, you would play as player 1 against the computer. If you select with "Enter" key, you would play as player 2 against the computer.
